# Coding Kinetics Workshop

## What you will Learn
- Understand how HTML & CSS work together
- Build a basic webpage from scratch
- Publish webpage on GitHub to start your online code  portfolio
- Leave with confidence to continue their coding journey

## What you will need
All the following programs are free to sign up for, and download.

- A Github account (sign up [here](https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Fsignup))
- A Integrated Developer Environment (IDE). 
  - Recommended IDEs to choose from:
    - [WebStorm IDE Community Edition](https://www.jetbrains.com/webstorm/promo/?source=google&medium=cpc&campaign=AMER_en_US-CST_WebStorm_Branded&term=webstorm&content=717267885243&gad_source=1&gad_campaignid=9641686287&gbraid=0AAAAADloJzifRbveGCFc5OBJwGRd3XNe8&gclid=Cj0KCQjwzaXFBhDlARIsAFPv-u9K1kk1jTcssVaaC0BRmciqBs9L_7cXiL0lGRwjfbzUbITf2ps1txcaAhhDEALw_wcB)
    - [Sublime Text](https://www.sublimetext.com/)
